## 📝 Deskripsi Program

Website ini merupakan aplikasi katalog produk sederhana yang dibangun menggunakan framework **Django**. Aplikasi ini bertujuan untuk menampilkan daftar produk beserta detailnya dalam tampilan web yang interaktif dan user-friendly.

Fitur utama dari website ini meliputi:

* **Homepage** → menampilkan halaman utama dengan deskripsi singkat
* **Halaman Daftar Produk** → menampilkan beberapa produk dalam bentuk katalog
* **Halaman Detail Produk** → menampilkan informasi lengkap dari produk berdasarkan ID
* **Halaman Kontak** → menampilkan informasi kontak

Data produk pada aplikasi ini masih menggunakan metode **hardcoded (tanpa database)** sehingga fokus utama berada pada implementasi routing, views, dan template di Django.

## 🚀 Cara Menjalankan Program

1. Clone repository:

   ```bash
   git clone https://github.com/AdindaMuarriva/Katalog.git
   ```

2. Masuk ke folder project:

   ```bash
   cd Katalog
   ```

3. Jalankan server:

   ```bash
   python manage.py runserver
   ```

4. Buka di browser:

   ```
   http://127.0.0.1:8000/
   ```
